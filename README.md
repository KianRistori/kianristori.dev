## kianristori.dev

- **Framework**: [Next.js](https://nextjs.org/)
- **Deployment**: [Vercel](https://vercel.com)
- **Content**: [MDX](https://github.com/mdx-js/mdx)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)


## Overview

- `data/*` - MDX data that is used for blogs, newsletters, and code snippets.
- `components/*` - Different components: PostCard, Navbar, Footer ... 
- `lib/*` - Short for "library", a collection of helpful utilities or code for external services.
- `pages/api/*` - [API routes](https://nextjs.org/docs/api-routes/introduction) post views.
- `pages/blog/*` - Static pre-rendered blog pages using MDX.
- `pages/projects/*` - Static pre-rendered project pages using MDX.
- `pages/*` - All other static pages.
- `public/*` - Static assets including fonts and images.
- `styles/*` - A small amount of global styles. I'm mostly using vanilla Tailwind CSS.

## Running Locally

```bash
$ git clone https://github.com/KianRistori/kianristori.dev.git
$ cd leerob.io
$ npm install
$ npm run dev
```
